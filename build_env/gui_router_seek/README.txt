Utilização do debugger (versão lite)

python3 main.py testcases/<nome_do_projeto>.yaml

Dentro do testcases tem que ter:
** Arquivo <nome_do_projeto>.yaml
** pasta <nome_do_projeto> com os logs da brNoC

Exemplo:
 testcases/dtwIOdsz.yaml
 testcases/dtwIOdsz/<addr>.json

Dependências:
 BTrees
pip3 install BTrees

 TKinter
yum install python3-tkinter
